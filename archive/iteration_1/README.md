# Iteration 1 Code Archive

This directory contains code and documentation from Iteration 1.

Iteration 1 implemented autonomous workflows, semantic merging,
deduplication, and JSON-based persistence.

Iteration 2 explicitly replaces these approaches with:
- Explicit CLI commands
- Human-controlled Excel workflows
- No autonomous consolidation or deduplication
- Strong Human Primacy enforcement

This code is archived for historical and audit reference only.

⚠️ This code MUST NOT be reintroduced into active paths
without explicit human review and approval.
